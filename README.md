# Finance-Dashboard-UI
Finance Dashboard (Streamlit)
This project is an interactive finance dashboard built with Streamlit and Plotly. It demonstrates how to design a user interface, manage application state, and provide insights into financial activity using mock data.

Features
- Theme toggle between light and dark mode

- Summary cards showing total balance, income, and expenses

Interactive charts:

- Line chart for balance trend over time

- Pie chart for spending breakdown by category

- Bar chart for monthly spending comparison

- Transactions table with search and sorting by date or amount

Role-based interface:

- Viewer role can only view data

- Admin role can add new transactions

- Insights section highlighting highest spending category and projected monthly expenses

- Export functionality to download transactions as CSV
